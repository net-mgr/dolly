## Dolly
YAMAHA RTX1200の設定ファイルを取得して，指定したファイルに保存する．

## Files
- dolly.env.sample

dolly.sh内で使用する環境変数を設定するためのファイルのサンプル．使用する際はこのファイルをコピーして，dolly.envとしておく．

- dolly.sh

dolly.envで指定した環境変数を基に，ルータの設定ファイルを取得し，保存するスクリプト

## Settings

dolly.env.sampleをdolly.envにコピーし，環境変数を設定する．

```
% cp dolly.env.sample dolly.env
```

dolly.envでは以下の環境変数を設定する．

-----------
NUM\_OF\_GW:
設定ファイルを取得したいルータの個数

GW\<x\>:
設定ファイルを取得したいルータ(x個目)のホスト名

GW\<x\>\_CONF:
ルータ(x個目)の設定ファイルを配置する計算機上のパス

-----------


## Usage
```
% bash dolly.sh
```

本スクリプトは/root/dolly以下に設定ファイルが存在することを想定している．
ルータの設定ファイルを取得して，指定したファイルに保存する．
ルータの設定ファイルは tftp によって取得しているため，必要であればルータの設定を行う必要がある．
